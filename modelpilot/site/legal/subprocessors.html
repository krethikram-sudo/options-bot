Subprocessors

The third-party services ModelPilot relies on to deliver the product. We'll keep this page current and give notice of material changes.

Important: in the default self-hosted / thin-client deployment, your prompt content goes directly from your infrastructure to Anthropic using your own API key — it does not pass through ModelPilot, so Anthropic is your model provider, not ModelPilot's subprocessor for prompt content. In the optional hosted-gateway deployment, request content is processed in memory by ModelPilot only to route it (read by our classifier to select a model) and is not persisted. Except for that transient routing in hosted mode, the services below process only account data and aggregate/opt-in metadata.
