# Sparring (working codename)

**One-liner:** Evaluate candidates by watching them debate Claude — adversarial case-study
and behavioral scenarios that reveal how someone actually thinks, argues, and holds (or
updates) a position under pressure.

**Origin insight:** While prepping for Staff PM interviews using Claude, the highest-signal
moments weren't Q&A — they were the debates. When Claude took a strong position and pushed
back, you could see reasoning quality, intellectual honesty, conviction calibration, and
adaptability in a way no take-home or panel interview surfaces. Companies should be able to
buy that signal.
